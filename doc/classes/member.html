    A class for CAP members

    It holds all the functions, and data for members. The amount of Data held is based on the init levels

    init levels : -1= all from input 0=capid 1=capid+name+gender+achievement 2=1+text+member_type+picture 3=2+dates 4=3+emergency+unit
