------------------------------Installing-----------------------------------------------
To install this on a linux system follow these steps:
1. Extract this tar-ball to a convenient location.
2. Navigate to the extracted files. 
3. Ensure that a LAMP server stack is installed as well the php mysql native driver (php-mysqlnd)
4. Run install.sh as root (i.e. sudo bash install.sh)
5. Follow the onscreen instructions

For more details go to <https://code.google.com/p/squadron-manager/wiki/install>

----------------------------Overview----------------------------------------------------
Squadron Manager is a Local intra-net server meant to help maintain Civl Air Patrol
records at the Squadron Level. Currently Squadron Manager (squadMan) is focused
on tracking Cadet Promotions, event attendance, and testing and promotion requests.
SquadMan should be only used on the local network, and accessible by all squadron Computers.
